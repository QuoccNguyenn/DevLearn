## Hướng dẫn chạy project
# Em chào thầy/cô, để chạy project này, thầy cô vui lòng làm teo những bước sau đây:
- Chạy server bằng cách cd vào thư mục 'server' và chạy lệnh 'npm run-script server'.
- Chạy client bằng cách cd vào thư mục 'client' và chạy lệnh 'npm start'.
- Sau đó truy cập vào localhost:3000 để truy cập giao diện.
